## 1. 项目概述

Nexus AI 前端是一个响应式新闻阅读 SPA，优先适配移动端（max-width 480px），桌面端（≥768px）切换为侧边栏导航 + 960px 内容区双列卡片布局。主要功能：

- 用户注册 / 登录 / 登出
- 按数据源平台过滤浏览新闻，支持无限滚动
- 新闻详情阅读（优先显示中文翻译）、相关新闻跳转、阅读原文链接
- 收藏 / 历史记录管理
- 用户资料编辑与密码修改
- 头部刷新按钮触发后台采集

所有数据通过 REST API 与后端（FastAPI）交互，Token 存于 localStorage。

## 4. 路由设计

模式：HTML5 History（`createWebHistory`）

| 路径 | 组件 | 是否公开 |
|------|------|----------|
| `/login` | LoginView | 是 |
| `/register` | RegisterView | 是 |
| `/` | - | - 重定向至 `/news` |
| `/news` | NewsView | 否 |
| `/news/detail/:id` | NewsDetailView | 否 |
| `/profile` | ProfileView | 否 |

未登录访问受保护路由自动跳转 `/login`。所有路由组件懒加载。

## 5. 状态管理

### authStore

| 字段/方法 | 说明 |
|-----------|------|
| `token` | 认证 Token，初始从 localStorage 读取 |
| `userInfo` | 当前用户信息 |
| `isLoggedIn` | computed，是否已登录 |
| `login / register / logout` | 登录/注册/登出 |
| `fetchInfo / updateInfo` | 刷新/更新用户信息 |

### newsStore

| 字段/方法 | 说明 |
|-----------|------|
| `categories` | 数据源列表 |
| `activeSource` | 当前选中数据源（默认 `all`） |
| `newsList / page / hasMore / loading` | 列表状态 |
| `loadCategories()` | 加载数据源（有缓存则跳过） |
| `loadNews(source, reset?)` | 加载新闻，`reset=true` 重置列表 |
| `setCategory(id)` | 切换数据源并重载列表 |

## 6. API 接口对接

### 响应格式约定

```json
{ "code": 200, "message": "success", "data": { ... } }
```

`client.ts` 统一判断 `code !== 200` 时抛错。

### NewsDetail 类型

```typescript
interface NewsDetail {
  id: number
  title: string;      titleZh: string | null
  description: string | null;  descriptionZh: string | null
  content: string;    contentZh: string | null
  image: string | null
  author: string | null
  source: string | null
  sourceUrl: string | null
  publishTime: string
  views: number
  relatedNews: NewsItem[]
}
```

### 新闻接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/news/categories` | 数据源列表 |
| GET | `/api/news/list` | 新闻列表（`source/page/pageSize`） |
| GET | `/api/news/detail?id=` | 新闻详情 |
| GET | `/api/news/search?keyword=` | 关键词搜索 |
| POST | `/api/news/refresh` | 触发后台采集 |

## 7. 页面设计

### NewsView — 新闻列表页

- 顶部栏：Logo + LIVE 指示 + **刷新按钮**（点击触发后台采集，旋转动效，成功变绿）
- 数据源 Tab 栏（吸顶，横向滚动）
- 第一条：Hero 大卡片（封面图 + 文字覆盖层）
- 后续：标准卡片（缩略图右置）
- 桌面端（≥768px）：卡片区 CSS Grid 2 列，Hero 跨全宽
- 无限滚动加载，骨架屏动效，加载完显示 "END OF FEED"

### NewsDetailView — 新闻详情页

- 顶部栏：返回按钮 + ARTICLE 标签 + 收藏按钮
- 封面大图（有时无图）
- 内容显示优先级：`contentZh` → `descriptionZh` → `content` → `description`
- 标题优先显示：`titleZh` → `title`
- 底部"阅读原文"链接（`sourceUrl`，新标签页打开）
- 相关新闻列表（点击跳转，已修复同组件路由切换不刷新的 bug）
- 桌面端阅读宽度限制为 700px 居中
- 进入页面自动记录浏览历史

### ProfileView — 个人中心

- 头像（用户名首字母）、昵称、用户名
- 编辑资料 / 修改密码（底部 Sheet 模态框）
- 登出
- Tab：收藏 / 历史，支持分页、单条删除、清空

```css
--brand: #C8860A;          /* 琥珀色主色调 */

--text-secondary: #4A4540;
--text-muted: #8C8580;

--nav-h: 60px;
/* 数据源色 */
--hn: #E05D00;   --openai: #0D8A6A;
--google: #1A73E8;  --mit-fg: #C0364D;
```

### 字体

| 字体 | 用途 |
|------|------|
| Libre Baskerville | Logo 标题 |
| JetBrains Mono | 标签、元数据、等宽文字 |
| Noto Serif SC | 文章标题、卡片标题 |
| Noto Sans SC | 正文、界面文字 |

### 响应式布局

| 断点 | 布局 |
|------|------|
| < 768px（移动端） | 底部导航栏，单列卡片，max-width 480px |
| ≥ 768px（桌面端） | 固定左侧边栏（80px），内容区 max-width 960px，双列卡片网格 |

### 设计风格

Editorial / 纸质感，paper grain 纹理叠加（SVG filter，opacity 0.4），避免通用 AI 设计模板。
